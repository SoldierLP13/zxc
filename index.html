Welcome to zxc!

zxc is this super useful page on which you can view documentation for ModPE. ModPE doesn't have perfect documentation, so zxc was created to fix that.

You can navigate zxc by using the navigation on the left (or on the top, you mobile users.)

Download BlockLauncher to get started patching and testing scripts.

There's also a free version if you prefer ads and limitations.

Get

You can contribute to the documentation by forking the repo and editing the data/docs.json file.

Be sure to contribute valid JSON!
